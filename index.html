Work

I have completed several projects based around C#, my preferred coding language of choice. My most recent project is an Asp.Net MVC Web App built using Visual Studio Community. This app uses all C.R.U.D. functions to store the data in a Sql database. In order to showcase multiple NuGet packages, it uses both Entity, and Dapper for queries and executions to the database. I used the Microsoft Identity platform for user authorization, and the Front-End uses not only Html and CSS, but uses many bootstrap functions for scaleability between platforms and devices.

Before that I completed a console app that called and utilized APIs and JSON files for two primary purposes. The first was to simulate a humorous conversation between Kanye West and the fictional character Ron Swanson from the NBC Show "Parks and Recreation". It generates dialogue by making a series of calls using specific APIs for each character and assembles them in an amusing conversation fashon. The second part used the openweathermap.org API to retrieve the current temperature for the zip-code submitted by the user via input. Both of these utilize RESTful APIs for their operation.

In addition, I also have made a program called the Taco Parser project that uses the global coordinates of Taco Bell locations within a CSV file and calculates the two locations located farthest away from eachother. This was created using object oriented programming and test driven development to achieve the expected result.

About

It is time for me now to complete my training, and begin performing the tasks that are needed by a company to increase their efficency and quality by building quality software and maintaining their various systems.
